# 🚀 Summary & Quiz Generator

An AI-powered web application that automatically generates concise summaries and interactive quizzes from user-provided text, documents, or study materials. Designed to help students, educators, and professionals learn faster and retain information more effectively.

## 📌 Features

- 📄 Generate concise and accurate summaries from large text content
- 🧠 Automatically create multiple-choice quiz questions
- 🎯 Instant evaluation and scoring system
- ⚡ Fast AI-powered content processing
- 📱 Responsive and user-friendly interface
- 🔍 Improves learning, revision, and knowledge retention
- 📊 Real-time quiz performance tracking
- 🌐 Easy-to-use web-based platform

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Aryan-2314/Summary-Quiz-Generator.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd Summary-Quiz-Generator
```

### 3️⃣ Install Dependencies

For Frontend:

```bash
cd client
npm install
```

For Backend:

```bash
cd server
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_api_key
```

### 5️⃣ Start the Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```
